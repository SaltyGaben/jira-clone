```bash

npm install

# Using pnpm
pnpm install

# Using yarn
yarn install

# Using bun
bun install
```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Supabase credentials:

```env

SUPABASE_KEY=your_supabase_anon_key
```
